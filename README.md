# WebGuard Lite

## Overview

WebGuard Lite is a web security application developed using Spring Boot. It helps users check website URLs and identify suspicious or potentially malicious links.

## Features

* URL security analysis
* Phishing website detection
* User-friendly interface
* Fast result generation
* Secure web application architecture

## Technologies Used

* Java
* Spring Boot
* Maven
* HTML
* CSS
* Thymeleaf
* IntelliJ IDEA

## Project Structure

src/main/java

* HomeController.java
* WebGuardLiteApplication.java

src/main/resources

* templates/index.html
* templates/result.html
* application.properties

## Installation Steps

1. Clone the repository

git clone https://github.com/raseena-zeelan/webguard-lite.git

2. Open in IntelliJ IDEA

3. Install Maven dependencies

4. Run WebGuardLiteApplication.java

5. Open browser

http://localhost:8080
